
# Notes Taking App

A simple and interactive Notes Taking App built using HTML, CSS, and JavaScript. This project allows users to create, edit, and manage their notes efficiently through a clean and user-friendly interface.

## Features

* Add new notes instantly
* Edit existing notes
* Delete notes when no longer needed
* Toggle between edit and preview modes
* Dynamic user interface powered by JavaScript
* Lightweight and responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

Through this project, I improved my understanding of:

* DOM manipulation

* Dynamic element creation
* JavaScript functions and logic building
* Managing user interactions efficiently
